# shiki
Shiki is the rust-like programming language.


## Setup
```
$ git clone https://github.com/mizdra/shiki
$ cd shiki
$ cargo build
```


## Usage
```
## REPL
$ cargo run --bin shiki

## Test
$ cargo test
```
